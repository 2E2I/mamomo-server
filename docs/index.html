1.1. 전체 리스트

요청

파라미터 없이

GET /api/campaigns HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080

모든 파라미터, 헤더 지정

GET /api/campaigns?page=0&size=3&sort=due_date%2CDESC&category=1 HTTP/1.1
Content-Type: application/json;charset=UTF-8
Authorization: Bearer eyJhbGciOiJIUzUxMiJ9.eyJzdWIiOiJ0ZXN0QGVtYWlsLmNvbSIsImF1dGgiOiJST0xFX1VTRVIiLCJleHAiOjE2NDk4NTA1OTF9.1kcLnTHUNl2PcUWObgtxZs-3pnaDaUR-qkPMgct5MpqkA6v0yCYYWl4DxbdNR-n76KTBpBbRFzcdyoG7xVacSw

GET /api/campaigns?heart=true HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080

모든 캠페인 API 에 모두 적용되는 요청 헤더 입니다.

Name Description

Authorization

토큰 앞에 'Bearer '을 붙인다.인증 되었을 경우 유저에 따른 isHeart 여부가 표시된다.
