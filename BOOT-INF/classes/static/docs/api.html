개요

소개

캠페인

캠페인 API는 캠페인 리스트를 반환합니다.

- 전체 리스트

요청

GET /api/campaigns HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080

정렬 추가 시

GET /api/campaigns?sort=start_date%2Cdesc HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080

- 카테고리 리스트

요청

GET /api/campaigns?category=%EC%9A%B0%EB%A6%AC%EC%82%AC%ED%9A%8C HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080

정렬 추가 시

GET /api/campaigns?sort=start_date%2Cdesc&category=%EC%9A%B0%EB%A6%AC%EC%82%AC%ED%9A%8C HTTP/1.1
Content-Type: application/json;charset=UTF-8
Host: localhost:8080
